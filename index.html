2025.09.15 → 2025.10.21 · Fullstack Developer

모바일 신분증 도입 프로젝트

하나저축은행 원큐앱 내부에 모바일 신분증을 이용한 인증 방식을 추가 도입하는 개발. 기존 실물 신분증 단독 방식에서 확장하여 고객 편의성을 높이는 인증 채널을 구축했습니다.

Background / Problem
  • 기존 인증 방식이 실물 신분증 하나로 제한되어 고객 불편을 야기
  • App2App 방식은 네이티브 개발자 2명 공수 필요, 앱 심사로 운영 배포에 제약 발생
Solution / Process
  • 개발 공수·운영 유연성·대응속도를 고려해 App2App 대신 Web2App 방식을 자처하여 구현
  • OS별로 상이하게 동작하는 브릿지를 각 OS 네이티브 개발자와 협력하여 통합
Result
  • 앱 심사 없이 즉각 대응 가능한 운영 배포 환경 구성
  • 네이티브 2명의 공수 없이 웹 개발자 1명으로 운영 개발 가능
  • 인증·개발 흐름을 웹으로 통일해 개발 난이도 낮춤
  • 개발 기간 내 모바일 신분증 프로젝트 완성적으로 도입 완료

채널별로 각각 작성해야 했던 공지사항·팝업 등록 방식을 공통 템플릿 구조로 개선해, 현업의 반복 업무를 줄이고 운영 효율을 높인 프로젝트.

Background / Problem
  • 공지사항·팝업 등록 시 채널마다 작성 방식이 달라, 같은 내용을 각각 반복 입력해야 함
